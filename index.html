Tree Visualizer

Visualize LeetCode tree problems

How to Use

  • Single Values: [1,2,3,null,5] - Level-order with null for missing nodes
  • Multi-Values: [[1,2],[3,4],[5,6]] - Each node contains two numbers
  • N-ary Tree: [1,null,2,3,4,null,5,6] - Set max children and use null as separator
  • Graph: Same format, but cycles will be detected and shown with red dashed lines
